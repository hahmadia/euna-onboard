# Euna Payments Repository Abbreviations

When referencing any of these repositories by name or abbreviation, use the corresponding local path under `~/code/`:

- **NFE** / Navigate Frontend → `~/code/navigate_frontend`
- **RevM** / Revenue Management Dashboard / RMD → `~/code/revenue_management_dashboard`
- **Citizen Dash** / Citizen Dashboard → `~/code/citizen_dashboard`
- **PAPI** / Payment API → `~/code/payment_api`
- **cb-components** / @thecb/components → `~/code/cb-components`
- **POS** / Cashiering → `~/code/cb_pos` (backend) / `~/code/pos-frontend` (frontend)
- **KI** / Kiosk Interface → `~/code/kiosk_interface`
- **GQL** / GraphQL Service → `~/code/graphql_service`
- **GQLI** / GQL Interface → `~/code/gql_interface`
- **FLS** / File Lookup Service → `~/code/file_lookup_service`
- **DPS** / Device Payment Service → `~/code/device_payment_service`
- **DD** / Device Drivers → `~/code/device_drivers`
